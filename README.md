Android Mobile (Chrome Browser) ke liye:
Site Open Karein: Apne mobile mein Chrome kholiye aur apni Streamlit site ka URL (jo https://xxxx.streamlit.app karke hai) daaliye.

Menu mein Jayein: Upar right side mein jo 3 dots (⋮) hote hain, un par click karein.

Install App / Add to Home Screen: Wahan aapko ek option dikhega "Install app" ya "Add to Home screen". Use select karein.

Confirm Karein: Ek popup aayega jisme aap App ka naam (jaise: MarketMaster) likh sakte hain. "Add" par click kar dein.

Check Karein: Ab apne mobile ki Home Screen par jaiye. Wahan aapko terminal ka icon dikhega. Jab aap use click karenge, toh wo bina browser ke address bar ke ekdam full-screen app ki tarah khulega.
